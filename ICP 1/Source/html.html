Creation of Github Repository

Git Basic Tasks

  • Create Github Account
  • Create a new repository
  • Clone repository in Github Desktop
  • Create folders in local repository
  • Push and Pull to remote

  1. Create Github Account

    2. Create a Github account by assigning Username, E-mail and Password. Click on Sign Up for Github.

  2. Create a Repository

    3. Click on New Repository. Give a name for the repository. Make it as a public and initialize with a README file.

  3. Clone Repository

    4. Clone the repository using HTTPS link in the Github Desktop . We can see the local repository in Local Github.

  4. Create Folders

    5. Create Source and Destination folders in the local repository. Put the files in the Source folder and images in the Destination folder.

  5. Push and Pull to remote

    6. After adding files to the local repository, commit the changes. Push the changes to the remote github. Sync local with the remote.
